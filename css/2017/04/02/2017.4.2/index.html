这个星期天还不错

By ppdwg

发表于 2017-04-02

庆祝我的个人博客正式创建完成

首先，真的很谢谢这些天帮助我的三位大神。我前几天搭建好一个基本的博客之后，本以为就能正常的使用了，但是我在接下来修改主题的时候发现，我的本地博客无法更新到网上，虽然知道是个小问题，令我吃惊的是，我发现不了问题到底出在了哪儿，因为我对 Hexo 了解的实在太少了。我在网上看了很多人家解决相关问题的经验，但没有对我的问题有效的。为了解决这个问题，我花了很多的时间，在周六中午过后我就开始着手解决这个问题，当时我是预计几个小时就可以完成了，一低头敲代码，再一抬头，外面的天已经黑了，起身，感觉是晕晕沉沉的，问题却没有能解决掉，这是最令人不开心的。那天晚上吃完晚饭后，我又尝试去读官方的文档，作用确实很大，虽然最后并没有解决我的问题，却让我知道了很多相关的知识，让我知道了我写的哪些东西是对的。排除了很多问题，却看不到接下来还有多少问题在等着我去检查。星期天上午，我差不多八点才爬起来，周六晚上实在有点累。洗脸刷牙完毕，继续搞事情。中午我问了我北大的一个老同学，他给了不少建议，其中一个就成功了（其实我在周六就有试过这个方法，而且不止一次，就是不行，今天不知道为什么就成功了。解决 bug 真的是需要天时地利人和的啊）。

我的春游

我们现在算是放清明假了，春天的天这么好，有假期就应该出去走走的嘛。本以为可以和同学一起骑小黄车去浪，因为有个我们班别的宿舍的人在班群提及，我看没有太多人回应，我为表示积极响应，主动提出个人出游的意见，他们宿舍有人回复了句 “随便你” ，真心是有些尴尬。今后我也不太想响应班里的活动了，有任务就做，没事儿就回到属于我的世界里，我也有我的圈子，我的好朋友，宁愿多花时间去找我的朋友，看我喜欢的东西，也不愿去瞎凑合对我自己不热心的人和事了。今天去找浩子（北大的好朋友）了，算是我的春游吧，北大确实一年四季都很好看，它有它自己的文化。其实去哪儿无所谓，重要的是和谁一起，和浩子聊了一个下午，走了一个下午，感觉还是很棒的。走在未名湖边，虽然人不少，却感觉心很安静。没有对未来的担忧，没有对琐事的困扰，很喜欢鱼在游动的样子，说到这里，我好想看看真实的海豚长什么样，一定很漂亮吧。

继续我的阅读

今天过去了，那之前的已经不是很重要了，但总要留下些什么。我还是喜欢阅读，不管是纸质的书，还是网文，都是很不错的。人家都说读史明志，我也是类似，我超级喜欢读别人的故事。这其中也是有选择性的，我喜欢读淳朴一些的，偶尔也会读别的东西，但都是很有意思的文章。我觉得阅读一定会给我带来很多很多的东西，会让我写的东西越来越有内容，会让我的生活很充实、很快乐，毕竟，任何行为归根结底，都是为生活服务的。 我有个愿望，当我有所理解的时候，很容易就有个人可以听听我的分享。
